## 리파지토리 생성

## 리파지토리 초기화 및 커밋
> 저장소 초기화, 파일 추가 그리고 커밋하기

```
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:thinkwave/thinkwave.github.io.git
git push -u origin main
```
